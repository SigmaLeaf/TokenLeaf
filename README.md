# CryptoInsights Cryptocurrency Price Tracker

# Backend Server
## Installation
```
pip install virtualenv
virtualenv venv
source venv/bin/activate

pip install ccxt
pip install flask
pip install -U flask-cors
```

## Running
```
cd server
python src/server.py

open browser at localhost:5000
```
